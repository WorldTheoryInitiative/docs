## Get started in three steps

Start building with the Purity Router API and make your first API call.

### Step 1: Get your API key

<AccordionGroup>
  <Accordion icon="key" title="Sign up for an API key">
    Visit our [API keys page](https://purity.com/api-keys) to create your account and get your API key.
    
    <Tip>Keep your API key secure and never share it publicly.</Tip>
  </Accordion>
  <Accordion icon="shield" title="Understand authentication">
    All API requests require authentication using a Bearer token in the Authorization header:
    
    ```bash
    Authorization: Bearer YOUR_API_KEY
    ```
  </Accordion>
</AccordionGroup>

### Step 2: Make your first API call

<AccordionGroup>
  <Accordion icon="terminal" title="Test the API">
    Try making a request to get categories:
    
    ```bash
    curl -X GET "https://api.purity.com/categories-v1" \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
    
    <Check>You should receive a JSON response with available categories.</Check>
  </Accordion>
  <Accordion icon="code" title="Use our code examples">
    We provide examples in multiple programming languages. Check the [API reference](/api-reference/introduction) for examples in:
    
    - cURL
    - Python
    - JavaScript
    - Go
  </Accordion>
</AccordionGroup>

### Step 3: Explore the API

<Accordion icon="rocket" title="Start building">
  Now you're ready to explore our comprehensive API endpoints:
  
  - **Categories**: Browse available prediction market categories
  - **Events**: Get detailed event information and outcomes
  - **Markets**: Access real-time market data and pricing
  - **Price History**: Retrieve historical data and trends
  - **Trending**: Discover popular and trending markets
</Accordion>
